# i3lock config
This isn't really necessary but I find it cool asf. 
## Pre-requisites
i3lock-color (installable thru the AUR)
## Installation
To install this, copy the i3lock program into the /usr/local/bin/ directory. Copy the "Calvin and Hobbes.TTF" file to ~/.local/share/fonts (create the directory as necessary), then run `fc-cache -f -v` to update the fonts.
